# Security Policy

##  Dataset Security
This repository contains **synthetic data only**.  
No real, personal, or sensitive information is included.

##  Sensitive Data
- No Personally Identifiable Information (PII)
- No real user data
- No production credentials or secrets

## Best Practices
- Do not add real datasets containing sensitive data.
- Avoid committing API keys, tokens, or passwords.
- Use environment variables for secrets if needed.
